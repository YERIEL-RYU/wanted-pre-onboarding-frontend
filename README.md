# 원티드 프리온보딩 프론트엔드 - 선발 과제

---

## 프로젝트 실행 방법
### git clone
```
git clone https://github.com/YERIEL-RYU/wanted-pre-onboarding-frontend.git
```

### yarn install
```
yarn
```

### 프로젝트 실행
```
yarn start
```
